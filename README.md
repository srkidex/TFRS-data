# Text Format RS-data

**(You need to know to run: Downloading Files, Zip / UnZip Files, Cut / Move and Copy / Pate Operations)**

Text Format RS is a NW.js web / js app that formats text via COPY / PASTE to:

1. small caps

2. ALL CAPS

3. Capitalize Sentances

4. Words Capitalized
---
5. Letters from RS (Serbian) Latin to Cyrillic

6. Letters from RS (Serbian) Cyrillic to Latin
---

# BONUS INFO BUTTON to tell YOU ALL THIS AGAIN:

all small- all letters in lowercase

ALL CAPS - all letters in uppercase

Capitalize Sentences - first letter of each sentence in uppercase, the rest in lowercase

Words Capitalized - first letter of each word in uppercase, the rest in lowercase

Serbian convertion -  Latin letters to Cyrillic (e.g. "a" to "а", "b" to "б", etc.)

Serbian convertion - Cyrillic letters to Latin (e.g. "а" to "a", "б" to "b", etc.)

## Installation

1. Download [NW.js](https://nwjs.io/) NORMAL edition and Extract ZIP

2. Download ZIP [TFRS-data](https://github.com/srkidex/TFRS-data/archive/refs/heads/main.zip)

3. Extract ZIP to "TFRS-data-main" folder (make if needed) inside NW.js main folder.

4. CUT / MOVE "package.json" from "TFRS-data-main" UP to the folder above. 

5. RUN NW.js app (.exe, .bin ...)
